Web Semantica

Elementos con significado

Ejemplos de etiquetas no semanticas: Div, Span
Ejemplos de etiquetas semanticas: Table, from, article

Header

Es el encabezado de una pagina, seccion o articulo. Incluye elementos como el logo, texto introductorio, menus de navegacion. Puede haber multiples headers en una pagina, pero no puede haber un header dentro de un footer, o de otro header.

Footer

Es el pie de pagina de un sitio o seccion. Puede haber multiples por pagina. Puede incluir informacion como:
  • Informacion de contacto
  • Enlaces de la RSS
  • Mapa del sitio
  • Creditos
  • Formulario de contacto

Nav

Define un conjunto de enlaces de Navegacion. No todos los links deben estar dentro de un Nav

Main

> Crea un cotenedor para el contenido principal de nuestra pagina. Este debe ser unico y no debe inluir contenido repetido de otras secciones,como por ej del sidebar, logos, etc. No puede ser hijo de etiquetas como: artcle, aside, footer, header o nav

Aside

Define el contenido relacionado indirectamente con el contenido principal, como por ejemplo un sidebar.Elementos que solemos poner en un aside:
  • Publicidad
  • Biografia el autor
  • Enlaces relacionados
  • Informacion de Perfil

Section

Define una seccion en una pagina donde se agrupa los contenidos relacionados, y normalmente incluyen un titulo
Una web se compone de varias secciones, cada una con su contenido y proposito

Article

Define un contenido independiente . El mismo puede ser compartido o reutilizado en otro sitio sin requerir contexto
Tambien suele incluir un titulo
Ejemplos de uso:
  • Un post de un blog
  • Un articulo de una revista
  • Comentario de un usuario (Rese√±a)
  • Tarjeta de producto

Puedo encontrar articulos dentro de secciones y secciones dentro de articulos. Un articulo puede tener multiples secciones. Un aseccion puede tener multiples articulos

Figure

Define elementos autocontenidos, como puede ser una imagen, un diagrama, una tabla o una foto
Dentro de ella , se suele utilizar la etiqueta "figcaption"para escribir contenido

Mark

Define un texto resaltado, normalmente
